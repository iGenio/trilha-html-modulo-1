Tags h1 a h6

-- Voltar --

As tags h1 a h6 servem para definir titulos na estrutura do HTML onde h1 representa o o texto com MAIOR tamanho e o h6 o MENOR

Tag p

-- Voltar --

A tag p é usada para definir um paragrafo, de forma que cada tag p cria um paragrafo em um testo

Exemplo:

<p>Paragrafo 1</p>

<p>Paragrafo 2</p>

Tag ol

-- Voltar --

A tag ol cria uma lista ordenada, ou seja, uma lista enumerada de 1 ate o valor do último item

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Tag ul

-- Voltar --

A tag ul cria uma lista não ordenada, ou seja, uma lista que so tem a marcação sem o numero, como defult vem uma bolinha

Exemplo:

  • Item 1
  • Item 2
  • Item 3

Tag li

-- Voltar --

A tag li é a tag de lista de item, a tag ol e ul nao funcionam por si só precisa de uma tag filha no caso a li para criar a lista de itens dentro da lista
